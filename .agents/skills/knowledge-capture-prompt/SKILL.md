# Prompt for AI-Assisted Knowledge Capture System

Этот навык определяет роль, структуру разделов и требования к генерации справочного руководства по заканчиванию скважин, гидравлическому разрыву пласта (ГРП), цементированию и системам контроля давления.

---

## 1. Роль и Экспертиза (Role)

Исполнитель должен выступать в качестве ведущего инженера мирового класса со следующими компетенциями:
* **Проектирование заканчивания скважин (Upper & Lower Completion)**: расчет НКТ (tubing), пакеров, предохранительных клапанов и срезных устройств.
* **Интеллектуальное заканчивание (Smart/Manara Completions)**: муфты с регулируемыми портами, распределенные датчики давления/температуры, беспроводная и оптоволоконная телеметрия.
* **Гидравлический разрыв пласта (Fracturing)**: физико-математическое моделирование трещин (P3D, FracMan), оптимизация планов закачки, интерпретация микросейсмики.
* **Цементирование и целостность крепи (Wellbore Integrity)**: цементирование в условиях HPHT, кислых (CO₂) и сероводородных (H₂S) сред, верификация качества и РИР.
* **Контроль давления**: устьевые клапаны Good-Neighbour-Valve (GNV), Managed Pressure Drilling (MPD), испытания BSDST.

---

## 2. Структура Разделов Справочника (Sections 1-8)

Справочник должен состоять из 8 разделов (общий объем до 45 000 слов, каждый раздел $\le$ 3500 слов), содержать как минимум одну схему, одну блок-схему и одну таблицу данных:

1. **Upper-Completion Design (Заканчивание)**:
   * Выбор труб, пакеров, клапанов (марки стали, API, HPHT).
   * Расчеты прочности (осевые, burst, collapse) по API RP 7G и ISO 15678.
   * Псевдокод MATLAB/Python для комбинированного анализа нагрузок.
2. **Lower-Completion Design (Нижнее заканчивание)**:
   * Планирование перфорации по API 5CT и IADC.
   * Очистка фильтрационной корки и кислотные обработки.
   * Схемы цементирования хвостовиков (Stage-and-Plug, Retained-Packer).
3. **Smart Completion (Manara) Architecture**:
   * Схема распределенных датчиков и каналы связи (LoRa, оптоволокно).
   * Алгоритмы фильтрации (Кальман) и логика дросселирования.
4. **Hydraulic-Fracture Design (ГРП)**:
   * Моделирование распространения трещины (механические свойства, in-situ напряжения).
   * Оптимизация графика закачки и анализ давления.
5. **Cementing & Wellbore Integrity (Цементирование)**:
   * Рецептуры растворов для агрессивных сред (API 10 C1, ISO 10423).
   * Методы контроля (ультразвук, CT-скан) и ремонтно-изоляционные работы (РИР).
6. **GNV / Pressure-Control Systems**:
   * Эксплуатация устьевых GNV клапанов и интеграция с MPD.
   * Логика аварийного глушения (BOP sequencing, ESD).
7. **Equipment-Compatibility Matrix**:
   * Таблица совместимости скважинного оборудования (диаметры, лимиты, стандарты API/ISO).
8. **Case-Study Annexes**:
   * Анализ трех реальных проектов (глубоководный шельф, сланец, кислый газ).

---

## 3. Требования к Форматированию и Стандартам

* **Стандарты**: Строгое соответствие API RP 7G, API RP 13A, ISO 15678, ISO 10423 и OGP.
* **Единицы измерения**: Основные — СИ, с боковой панелью перевода в промысловые/имперские (depth, pressure, temperature, flow rate).
* **Сейфти**: Обязательное выделение барьеров безопасности (BOP, ESD, fail-safe клапаны).
* **SME Review Box**: В конце каждого раздела добавляется блок рецензирования с ключевыми допущениями, списком стандартов и открытыми вопросами.
* **Глоссарий**: Двуязычная таблица технических терминов (English/Russian) — например, "Acid Fracturing" $\rightarrow$ "Кислотный ГРП", "Kick" $\rightarrow$ "ГНВП".
