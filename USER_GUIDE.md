# HADALBORE_LAB — User Guide / Руководство пользователя

This guide explains how to install, unlock, and use **HADALBORE_LAB** on your devices.
Это руководство объясняет, как установить, разблокировать и использовать **HADALBORE_LAB** на ваших устройствах.

---

# 🇬🇧 English Version

## 1. Accessing the Application

You can access and run the application in two ways:

### Method A: Online Web Version (PWA)
1. Open the official link in your browser: `https://nikolasmega.github.io/HADALBORE_LAB/`
2. The website will load and automatically cache all database records for offline use.

### Method B: Offline Local Run (ZIP Archive)
1. Download the `HADALBORE_LAB_v1.3.0.zip` file from the GitHub Releases page.
2. Extract the ZIP archive to any directory on your computer or USB drive.
3. Open the extracted folder and double-click **`index.html`** to launch the app instantly—no internet connection required.

---

## 2. Unlocking the Database

To protect intellectual property, the database is fully encrypted.
1. When you launch the application, you will be prompted with the **Unlock Screen**.
2. The default access key **`HADALBORE2026`** is pre-filled automatically for your convenience.
3. Click the **"Unlock"** button to decrypt the database and enter the platform.
4. Your decrypted state will be cached in the browser session. You will not need to re-enter the password when refreshing the page.

---

## 3. Installing the App on Devices (PWA)

You can install HADALBORE_LAB directly onto your phone, tablet, or PC desktop to launch it like a native application with an offline app icon:

* **iPhone & iPad (Safari)**:
  1. Open the website link in Safari.
  2. Tap the **Share** button (box with an arrow pointing up) at the bottom.
  3. Scroll down and tap **Add to Home Screen**.
* **Android Phones & Tablets (Chrome)**:
  1. Open the website link in Chrome.
  2. Tap the pop-up notification: *“Add HADALBORE_LAB to Home screen”* or tap the three dots in the top right and select **Install app**.
* **Desktop computers (PC / Mac)**:
  1. Open the website link in Chrome or Edge.
  2. Look at the right side of the address bar and click the **Install App** icon (looks like a monitor with an arrow).

## 4. Key Features & How to Use Them

### 🔍 Typo-Tolerant Search
- Use the search bar on the homepage or header to find any tubular, elastomer, standard, or connection.
- The search engine automatically tolerates typos, spelling mistakes, and matches aliases/synonyms.

### ⇄ Side-by-Side Comparison
- Click the **"Compare"** button on any record card to add it to the comparison queue.
- You can add up to 4 items (e.g., compare tubulars or steel grades side-by-side).
- Navigate to the **"Compare"** tab in the sidebar to view a clean comparative table highlighting technical differences.

### ⚙️ View Modes (Field View vs. Technical View)
- Open **Settings** (gear icon) to toggle between:
  - **Field View**: Optimized layout for quick diagnostics on-site, using sunlight-readable colors and simplified parameters.
  - **Technical View**: Displays advanced metallurgical formulas, strict envelope limits, and deep design guidelines.

### 💛 Feedback & Donations
- If you find any data or feature missing, click the **"Feedback"** button on the homepage to be redirected to GitHub to request updates.
- If the app is useful, you can support development by donating to the crypto address listed at the bottom of the `README` file.

## 4. Ключевые функции и использование

### 🔍 Поиск с защитой от опечаток
- Используйте поисковую строку на главной странице или в шапке сайта для поиска труб, эластомеров, стандартов или соединений.
- Поисковый движок автоматически исправляет опечатки и находит записи по синонимам и альтернативным названиям.

### ⇄ Сравнение элементов «Бок о бок»
- Нажмите кнопку **«Сравнить»** на карточке любого элемента, чтобы добавить его в очередь.
- Вы можете сравнивать до 4 позиций одновременно (например, разные трубы или марки стали).
- Перейдите в раздел **«Сравнение»** в боковом меню, чтобы увидеть таблицу сопоставления технических характеристик.

### ⚙️ Режимы отображения (Полевой vs Технический)
- Откройте **Настройки** (иконка шестеренки), чтобы переключить режим:
  - **Полевой режим (Field View)**: Контрастный интерфейс для быстрой работы на скважине, использующий крупные элементы и только самые критичные параметры.
  - **Технический режим (Technical View)**: Отображает детальные металлургические формулы, диаграммы предельных нагрузок и глубокие рекомендации по эксплуатации.

### 💛 Обратная связь и донаты
- Если вам не хватает каких-то данных или функций, нажмите кнопку **«Обратная связь»** на главной странице — вы будете перенаправлены на GitHub для создания запроса.
- Если справочник оказался полезным, вы можете поддержать проект донатом на криптокошелек, указанный в нижней части файла `README`.
