# Справочник резьбовых соединений RedBook (Technical Data Handbook 2002)

Этот справочник содержит технические характеристики, диапазоны моментов свинчивания и типы уплотнений для премиальных резьбовых соединений, перенесенные из официального каталога RedBook Technical Data Handbook 2002.

---

## 1. Hydril SuPreme LX (Страница 167)
* **Тип соединения**: Безмуфтовое интегральное в одну линию (Integral Flush)
* **Уплотнение**: Радиальное металл-металл (Metal-to-metal radial seal)
* **Диапазон моментов**: `2,400 - 5,800 ft-lbs`
* **Моменты по типоразмерам (Makeup Torque by Size)**:
  * **5.000"**: `2,400 - 3,100 ft-lbs`
  * **5.500"**: `2,900 - 3,700 ft-lbs`
  * **7.000"**: `4,900 - 5,800 ft-lbs`
  * **9.625"**: `7,500 - 9,300 ft-lbs`
