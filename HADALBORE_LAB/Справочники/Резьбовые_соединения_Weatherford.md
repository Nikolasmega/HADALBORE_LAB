# Справочник резьбовых соединений Weatherford (Technical Data Handbook 2002)

Этот справочник содержит технические характеристики, диапазоны моментов свинчивания и типы уплотнений для премиальных резьбовых соединений, перенесенные из официального каталога Weatherford Technical Data Handbook 2002.

---

## 1. VAM TOP / VAM TOP HT (Страница 188)
* **Тип соединения**: Муфтовое премиальное (Coupled Premium)
* **Тип уплотнения**: Металл-металл (Metal-to-metal seal) с упорным уступом (Torque Shoulder).
* **Описание**: Обладает высокой прочностью на растяжение и кручение. Отлично подходит для экстремальных HPHT сред и искривленных скважин (ERD).
* **Спецификации моментов свинчивания (Makeup Torque)**:
  - 5.000" (18.0 lb/ft): 422 - 475 kips joint strength (рекомендуемый момент свинчивания: `3,800 - 4,850 ft-lbs`).
  - 5.500" (20.0 lb/ft): `4,300 - 5,500 ft-lbs`.
  - 7.000" (29.0 lb/ft): `6,500 - 8,000 ft-lbs`.
  - 9.625" (47.0 lb/ft): `10,800 - 13,500 ft-lbs`.

---

## 2. Hydril SuPreme LX (Страница 167)
* **Тип соединения**: Безмуфтовое интегральное в одну линию (Non-upset, Integral Flush)
* **Тип уплотнения**: Радиальное металл-металл (Metal-to-metal radial seal).
* **Описание**: Гладкое снаружи и изнутри соединение. Разработано для минимизации затрубного пространства и обеспечения герметичности по газу.
* **Спецификации моментов свинчивания (Makeup Torque)**:
  - 5.000" (21.4 lb/ft): `2,400 - 3,100 ft-lbs`.
  - 5.500" (23.0 lb/ft): `2,900 - 3,700 ft-lbs`.
  - 7.000" (32.0 lb/ft): `4,900 - 5,800 ft-lbs`.
  - 9.625" (47.0 lb/ft): `7,500 - 9,300 ft-lbs`.

---

## 3. Hunting SEAL-LOCK HC (Страница 161)
* **Тип соединения**: Муфтовое (Coupled)
* **Тип уплотнения**: Металл-металл (Metal-to-metal seal).
* **Описание**: Предназначено для глубоких и HPHT скважин, обеспечивает высокую герметичность по газу.
* **Спецификации моментов свинчивания (Makeup Torque)**:
  - 5.000" (18.0 lb/ft): `2,900 - 3,600 ft-lbs`.
  - 5.500" (23.0 lb/ft): `3,500 - 4,400 ft-lbs`.
  - 7.000" (29.0 lb/ft): `5,100 - 6,300 ft-lbs`.
  - 7.625" (33.7 lb/ft): `5,800 - 7,200 ft-lbs`.
